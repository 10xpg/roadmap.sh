# Github User Activity CLI
The github activity cli is a project used to fetch the recent activity of a GitHub user and display it in the terminal.
This tool was created utilizing the <b>python3 standard library</b>.
- <b>project url: [task-cli](https://roadmap.sh/projects/task-tracker)</b>

## Prerequisites
- python3 >= 3.12.0
- cloned verison of this repository

## Usage
```zsh

python3 github-activity.py <github-username>
```
